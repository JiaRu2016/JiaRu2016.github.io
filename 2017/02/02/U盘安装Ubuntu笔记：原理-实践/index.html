U盘安装Ubuntu笔记：原理&实践

2017-02-02

Intro

最近打算开始折腾Linux系统，第一步当然是自己动手安装Ubuntu了。想把安装过程和涉及到的基本原理记录下来以备日后查询，于是有了这篇文章。

要在一台电脑上安装Ubuntu系统，总的来说分为两步：第一步是制作Ubuntu安装盘，第二步是使用做好的安装盘安装Ubuntu系统到一台电脑上。

制作Ubuntu安装U盘（Mac命令行方式）

这一步的目的是要把Ubuntu系统烧录到U盘中。这个制作好的启动盘可以多次用来装系统，因此建议保留这个U盘，万一将来一不小心rm -rf了，就可以直接用它来重装系统。也可以方便的给其他人的电脑装。

所以其实这一步不是必须的，你可以直接在技术群里吼一声“谁有Ubuntu启动U盘借我用下“或者干脆淘宝搜”Ubuntu系统U盘“，就可以跳过这一步了。。。这里我还是打算自己制作。

不同操作系统下制作Ubuntu启动盘的方式不同。如果是Windows系统可以用一个叫Universal USB Installer的软件，Mac系统则是用命令行dd命令。这里我用的是Mac命令行方式。

步骤

  1. 首先当然是去官网下载ISO镜像文件
  2. 然后使用hditutil命令把.iso文件转换成.dmg文件
  3. 插入U盘，先diskutil list查看你的U盘是哪个，然后diskutil unmountDisk卸载掉你的U盘。
  4. 使用dd命令烧录U盘，完成后弹出U盘

安装Ubuntu

插上U盘，开机启动时狂按F12进入BIOS（不同的笔记本进入BIOS方式不同，开机画面上一般有提示），选择从USB启动

欢迎界面。如果想先尝试一下选择Try Ubuntu，如果想直接安装选择Install Ubuntu。
